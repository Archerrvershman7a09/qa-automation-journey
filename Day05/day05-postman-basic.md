# Day 05: Postman Basic

## Main Goal

The goal of Day 05 is to learn how to use Postman for basic API testing.

I practiced sending HTTP requests, checking API responses, validating status codes, checking response fields, and exporting a Postman collection as a JSON file.

---

## 1. What is Postman?

Postman is an API testing tool.

QA engineers use Postman to manually send API requests and check if the server returns the correct response.

Simple flow:

```text
Choose method → Enter URL → Add headers/body → Send request → Check response → Run tests
