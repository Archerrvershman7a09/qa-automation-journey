# 90 Day Goals


## Month 1: Python + SQL + QA basics + QA Foundation
- Learn QA fundamentals
- Understand SDLC and STLC
- Practice test cases and bug reports
- Learn Git and GitHub basics
- Start SQL basics


## Month 2: API + Postman + Git + Test Design + Tools and API Testing
- Practice SQL queries
- Learn HTTP and REST API basics
- Use Postman for API testing
- Practice test design techniques
- Improve technical English


## Month 3: Automation and Portfolio + Interview Prep
- Learn TypeScript basics
- Start Playwright automation
- Write UI tests
- Write API tests
- Build a clean GitHub portfolio
- Prepare for junior QA / QA Automation interviews
