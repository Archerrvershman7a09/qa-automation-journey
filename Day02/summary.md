# Day 02 Summary

## What I learned

Today I learned the difference between SDLC and STLC.

SDLC is the full software development life cycle. It explains how a product is planned, designed, developed, tested, deployed, and maintained.

STLC is the software testing life cycle. It explains how QA plans, designs, executes, and closes the testing process.

I also learned the difference between Test Levels and Test Types.

Test Levels show where testing happens:
- Unit
- Integration
- System
- UAT

Test Types show what kind of testing is performed:
- Smoke
- Regression
- E2E
- Functional

## What was difficult

The most difficult part was understanding the difference between Test Levels and Test Types.

Now I understand it this way:

Test Level = where we test.

Test Type = what kind of testing we perform and why.
